<!-- GETTING STARTED -->
### Começando o projeto

#### Pré-requisitos
Ter o NPM instalado

* NPM
  ```sh
  npm install npm@latest -g
  ```

#### Instalação


* Clone o repositório
   ```sh
   git clone https://github.com/horodeski/HumildeTentativaDeFazerUmPortfolio
   ```

* Instale as dependências
  ```sh
  npm install
  ```

* Inicie o servidor de desenvolvimento

  ```sh
  npm run dev
  ```
